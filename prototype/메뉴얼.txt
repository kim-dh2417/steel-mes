========================================
   VICON - 데크 3공장 MES 프로토타입
          실행 메뉴얼
========================================

안녕하세요!
이 문서는 VICON MES 프로토타입을 처음 사용하시는 분을 위한
실행 안내서입니다.


[ 실행 방법 ]
──────────────────────────────────────

  별도의 설치나 서버 구동이 필요 없습니다.
  아래 한 가지 단계만 따라주시면 됩니다.

  1. 이 폴더 안에 있는 index.html 파일을 더블 클릭하세요.
     -> 기본 웹 브라우저에서 바로 실행됩니다.

  그게 전부입니다!


[ 권장 브라우저 ]
──────────────────────────────────────

  - Google Chrome (권장)
  - Microsoft Edge
  - Mozilla Firefox

  ※ Internet Explorer는 지원하지 않습니다.


[ 프로젝트 구성 파일 ]
──────────────────────────────────────

  prototype/
  ├── index.html   ... 메인 화면 (이 파일을 실행하세요)
  ├── styles.css   ... 화면 스타일
  ├── app.js       ... 화면 동작 로직
  └── 메뉴얼.txt    ... 지금 읽고 계신 이 문서

  세 개의 파일이 같은 폴더에 함께 있어야
  정상적으로 동작합니다. 파일을 옮기실 때는
  반드시 함께 이동해 주세요.


[ 주요 기능 ]
──────────────────────────────────────

  - 생산 대시보드 / 생산 현황판
  - 공정별 실적 조회 (신선, TG, 포밍, 조립)
  - 시간대별 생산 추이 확인
  - 그 외 다양한 MES 메뉴 탐색


[ 문의 ]
──────────────────────────────────────

  실행 중 문제가 발생하거나 궁금한 점이 있으시면
  담당자에게 편하게 연락해 주세요.
